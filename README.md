# Retinal Vessel Segmentation Using Deep Learning

This project was developed as part of my master's research and focuses on the automatic segmentation of retinal blood vessels from fundus images using deep learning techniques.

The work involves preprocessing retinal images, applying data augmentation, and training segmentation models including U-Net, ResU-Net, and attention-based variants. The models were trained and evaluated on the DRIVE and CHASE_DB1 datasets using metrics such as Accuracy, Precision, Recall, F1-Score, and Dice Coefficient.

The entire implementation, including data preparation, model training, evaluation, and results, is contained within a single Google Colab notebook.

## Technologies Used

- Python
- PyTorch
- OpenCV
- NumPy
- Matplotlib
- Google Colab

## Datasets

The project uses the following publicly available retinal image datasets:

- [DRIVE Dataset](https://drive.grand-challenge.org/)
- [CHASE_DB1 Dataset](https://blogs.kingston.ac.uk/retinal/chasedb1/)
